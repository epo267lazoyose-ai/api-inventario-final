# API Inventario TI

Este proyecto consiste en el desarrollo de una API REST para la gestión de inventario de dispositivos tecnológicos.

## Tecnologías
- Python
- Flask
- Flask-CORS
- Render (deploy)

## Endpoints

### GET /devices
Obtiene todos los dispositivos

### GET /devices/{id}
Obtiene un dispositivo por ID

### POST /devices
Crea un nuevo dispositivo

### PUT /devices/{id}
Actualiza un dispositivo

### DELETE /devices/{id}
Elimina un dispositivo

## Despliegue
API disponible en:
https://api-inventario-final.onrender.com

## Funcionalidades
- CRUD completo
- Validación de datos
- Manejo de errores HTTP
- Conexión con frontend
